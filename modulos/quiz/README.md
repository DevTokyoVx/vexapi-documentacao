# ❓ Módulo Quiz & Trivia: Vex API

O módulo **Quiz** da Vex API permite a geração dinâmica de perguntas e respostas para jogos de trivia, bots de entretenimento e plataformas educacionais.

---

## 🚀 Categorias Disponíveis

| Categoria | Rota | Descrição |
| :--- | :--- | :--- |
| **Animais** | `/api/quiz/animais` | Perguntas sobre o reino animal. |
| **Filmes** | `/api/quiz/filmes` | Cinema, atores e produções. |
| **Geografia** | `/api/quiz/geografia` | Países, capitais e curiosidades. |
| **História** | `/api/quiz/historia` | Fatos históricos e personalidades. |
| **Música** | `/api/quiz/musica` | Artistas, álbuns e gêneros. |
| **Tecnologia** | `/api/quiz/tecnologia` | Mundo tech e inovações. |

---
