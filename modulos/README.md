# 📦 Índice de Módulos: Vex API

Bem-vindo ao núcleo de exemplos da Vex API. Aqui você encontrará os scripts organizados por categorias funcionais.

---

## 📂 Organização das Pastas

Cada pasta abaixo contém exemplos práticos (`.js`) e um guia específico (`README.md`) para aquela categoria.

### 🎨 [Canvas](./canvas)
> Ferramentas para manipulação de imagens, criação de cards de boas-vindas, ranks e efeitos visuais dinâmicos.

### 📥 [Downloads](./downloads)
> Scripts para extração de mídias de alta qualidade de plataformas como TikTok, Instagram, YouTube, Facebook e Spotify.

### 🖼️ [Logos & Design](./logos)
> Geração de logotipos estilizados, identidades visuais e o famoso sistema de combinação de emojis (Emojimix).

### 🔍 [Pesquisas & Utils](./pesquisas)
> Ferramentas de busca para músicas (SoundCloud/Letras), informações de filmes, significados de nomes, horóscopo e mais.

### 📰 [News & Notícias](./news)
> Integração com os maiores portais de notícias do Brasil (G1, UOL, Estadão) para coleta de manchetes em tempo real.

### 🎮 [Quiz & Trivia](./quiz)
> Motor de perguntas e respostas para games e bots, com categorias variadas e níveis de dificuldade.

---

## 🛠️ Como executar os scripts

1. Navegue até a pasta do módulo desejado.
2. Execute o script usando Node.js:
   ```bash
   node nome-do-arquivo.js
   ```
